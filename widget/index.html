AI Receptionist Widget Demo

How to use:

1. Copy the widget script to your website:

<script>
window.AIReceptionistConfig = {
    apiUrl: 'http://localhost:8000',
    businessId: 'restaurant_001'
};
</script>
<script src="http://localhost:8000/widget/widget.js"></script>

2. Or load it directly:

<script src="http://localhost:8000/widget/widget.js"></script>
<script>
    window.AIReceptionist = new window.AIReceptionistWidget({
        apiUrl: 'http://localhost:8000',
        businessId: 'restaurant_001',
        position: 'bottom-right'
    });
</script>

The chat widget should appear in the bottom-right corner of this page.
